Football Kit Store Website

The Football Kit Store website started as a personal project with the potential to become a full e-commerce business. While it was never fully completed, it showcases the core design, product pages, and functionality. There are plans to make improvements and add fixes in the future, turning it into a more complete and polished platform.

View Live Project

Xstore

The Football Kit Store was created as a personal project to explore e-commerce design concepts, offering a simple and modern interface for browsing football jerseys.s

Built with HTML, CSS, and JavaScript, the site includes responsive layouts, a product carousel, and clean UI elements that mimic real online store functionality.

The project uses modular front-end structure, media queries, and reusable components, serving as a solid prototype for a fully developed e-commerce platform in the future.

Website Structure & Features

The slideshow below provides a visual overview of the website’s structure, showcasing how each section is organized and how users navigate through the main features.

Entry Interface

The Entry Interface features a clean navigation bar with dropdown menus, a search bar, and a shopping cart icon, offering easy access to all store sections. Beneath it, a dynamic slideshow showcases featured football kits and highlights the store’s main offerings.

"New Products" Section

The “New Products” section showcases the latest football jerseys in a responsive flex layout. Each item displays the front of the jersey, and when hovered, it smoothly switches to the backside, giving users a quick preview of the full design.

"Other Products" Section

The “Other Products” section highlights additional football jerseys, displaying them in an organized layout. Users can easily browse these items to explore the full range of available designs beyond the featured products.

"Product Information" Page

The “Product Information” page appears when a user selects a jersey, providing detailed information about the item. It includes selectable size buttons, a zoom-on-hover feature, and an interactive image gallery that lets users shuffle between different views, such as front, back, and side profiles of the jersey.
